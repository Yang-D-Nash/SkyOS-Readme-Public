## Owner Scope

Owner access in SkyOS can influence:

- runtime and kill-switch behavior
- AI prompts, providers, and quotas
- legal content settings
- role and quota assignment
- Shopify and payment configuration
- membership and revenue operations

## Operating Rule

Owner access is production authority, not a shortcut around process. If a live change affects
trust, billing, permissions, or legal disclosures, update the related documentation in the same pass.
