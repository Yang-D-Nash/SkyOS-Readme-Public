# SkyOS Release Checklist

## Build und Technik

- [ ] iOS Full Build gruen
- [ ] Android Full Build gruen
- [ ] Keine fehlenden Referenzen / Compilerfehler
- [ ] Keine offensichtlichen Runtime-Crashes im Smoke

## Produkt-Sanity

- [ ] Home oeffnet korrekt
- [ ] AI und Agent oeffnen korrekt
- [ ] Profile oeffnet korrekt
- [ ] Settings oeffnet korrekt
- [ ] Sheets/Dialoge lassen sich stabil oeffnen/schliessen

## Purchases und Membership

- [ ] Purchase-Entry oeffnet Store-Pfad
- [ ] Restore funktioniert
- [ ] Planstatus aktualisiert sich sichtbar
- [ ] Deny/Retry/Upgrade-Kontext wirkt ruhig und klar

## Legal und Vertrauen

- [ ] Legal Center in Settings vollstaendig (Terms, Privacy, Subscription, AI Notice, Impressum, Support)
- [ ] Support-Kontakt korrekt
- [ ] Last-Updated Angaben aktuell
- [ ] Externe Rechtspruefung dokumentiert

## Localization

- [ ] Keine kritischen fehlenden Keys in Default-Locale
- [ ] Tier-1 Texte geprueft (EN, DE, ES, FR, PT)
- [ ] Keine abgeschnittenen Labels in kritischen Screens

## Release Ops

- [ ] Store Assets/Metadata aktuell
- [ ] Crash/Analytics Monitoring aktiv
- [ ] Rollback-Plan vorhanden
- [ ] Go/No-Go dokumentiert
