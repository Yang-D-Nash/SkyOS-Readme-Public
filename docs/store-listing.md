# SkyOS Premium Store Listing

Status: `store-ready copy draft (legal finalization pending)`

This document contains production-grade listing copy for App Store and Google Play, optimized for premium branding, clarity, and conversion.

## Positioning

SkyOS is the operating core and visual system.  
Skydown is the product experience users install and use daily.

| Brand | Role in listing copy |
| --- | --- |
| `SkyOS` | system foundation, platform quality, premium identity |
| `Skydown` | app product for AI, Music, Video, and Merch |
| `ZweiZwei / 22` | music-specific brand context only |
| `Skydown x 22` | merch and commerce context only |

### Naming lock (must stay consistent)

- Use `SkyOS` for system/platform language and launcher identity.
- Use `Skydown` for app product communication (AI, Video, core UX).
- Use `ZweiZwei / 22` only for music surfaces and music copy.
- Use `Skydown x 22` only for merch and commerce surfaces.
- Do not swap or blend these labels in store metadata, screenshots, or release notes.

## Conversion Strategy

- Hook fast: first line explains user value in plain words.
- Show structure: AI, Music, Video, and Merch are one coherent system.
- Build trust: clear legal/support framing, no inflated claims.
- Keep premium tone: calm, modern, precise, no hype language.
- Preserve platform fit: App Store and Play Store copy adapted, not duplicated.
