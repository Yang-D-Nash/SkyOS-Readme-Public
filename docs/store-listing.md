# 22kydwn Premium Store Listing

Status: `store-ready copy draft (operator details finalized)`

This document contains production-grade listing copy for App Store and Google Play, optimized for premium branding, clarity, and conversion.

## Positioning

**22kydwn** is the umbrella brand and the name users install from the store.
**Skydown** is the operator behind the product. **Skydown Entertainment** and **ZweiZwei** are fused
sub-brands that stay visible inside the app and in listing copy where they help orientation.

| Brand | Role in listing copy |
| --- | --- |
| `22kydwn` | App Store / Play product name, launcher, platform promise |
| `Skydown` | Operator, support, legal, creator ecosystem |
| `Skydown Entertainment` | Video surfaces only |
| `ZweiZwei` | Music surfaces only |
| `Skydown x 22` / line merch marks | Merch and commerce only — existing artwork, no redesign in listing |

### Naming lock (must stay consistent)

- Use `22kydwn` for store title, launcher, screenshots, and the first two lines of every description.
- Use `Skydown Entertainment` for video — not a generic “video tab” without the line name.
- Use `ZweiZwei` for music — not “22 music” alone in primary copy.
- Use `Skydown x 22` (and existing merch line names) only for shop/merch paragraphs.
- Do not use `SkyOS` as a public product name in store metadata.
- Do not imply that merch logos were replaced; the app unifies the experience under `22kydwn`.

## Conversion Strategy

- Hook fast: first line explains user value in plain words.
- Show structure: AI, Music, Video, and Merch are one coherent system.
- Build trust: clear legal/support framing, no inflated claims.
- Keep premium tone: calm, modern, precise, no hype language.
- Preserve platform fit: App Store and Play Store copy adapted, not duplicated.
