# SkyOS Localization Terminology Glossary

This glossary defines mandatory wording for trust-critical UI copy across iOS and Android.

## Core Terms

- Membership
- Plan
- Upgrade
- Restore purchases
- Manage subscription
- Bot
- Agent
- Workflow
- Idle
- Retry
- Not available
- Support
- FAQ / Guide

## Usage Rules

- Use one canonical term per concept across all surfaces.
- Do not mix synonyms in the same flow (for example, avoid mixing "Subscription", "Membership", and "Abo" in one locale without clear design intent).
- Prefer calm, operational copy over marketing language.
- Keep CTA labels short for mobile safety.

## Tone Rules

- Calm
- Precise
- Trustworthy
- Non-pushy
- Human, not robotic

## Locale Notes

- Tier-1 locales (EN, DE, ES, FR, PT): premium-quality copy required in all primary and deep settings flows.
- Tier-2 locales (IT, NL, TR, PL, JA): maintain complete key coverage and high-quality fallback copy; prioritize short mobile-safe labels.
